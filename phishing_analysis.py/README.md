# Phishing Awareness Analysis

## Objective
The purpose of this project is to identify phishing attempts in emails by analyzing suspicious keywords, links, and common phishing indicators.

## Features
- Detects phishing-related keywords
- Extracts URLs from emails
- Identifies suspicious links
- Detects urgency tactics
- Generates a detailed report

## Technologies Used
- Python
- Regular Expressions (Regex)
- File Handling
- VS Code

## How to Run

Open terminal and run:

python phishing_analysis.py

## Output

The program:
1. Reads emails from sample_emails.txt
2. Analyzes phishing indicators
3. Displays results in terminal
4. Saves analysis to report.txt
