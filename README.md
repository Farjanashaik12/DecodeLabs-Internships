# DecodeLabs-Internships# Phishing Awareness Analysis

## Objective
Analyze emails and messages to identify phishing attempts.

## Features
- Detects suspicious keywords
- Identifies suspicious links
- Lists phishing red flags
- Explains why a message is unsafe
- Generates a report automatically

## Requirements
Python 3.x

## Run

python phishing_analysis.py

## Output
Results are displayed on the terminal and saved in report.txt.
