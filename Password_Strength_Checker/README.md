# Password Strength Checker

## Objective

The purpose of this project is to evaluate password security by checking various password characteristics and classifying passwords as Weak, Medium, or Strong.

## Features

- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special symbols
- Classifies password strength
- Generates a report file

## Technologies Used

- Python
- Regular Expressions (Regex)
- File Handling
- VS Code

## How to Run

python password_checker.py

## Output

Results are displayed in the terminal and saved in report.txt.
